Introduction

Robert Gleeson is a talented, self taught software engineer with over 8 years experience working in the software industry, both at the office and as a remote employee. In those 8 years, he has worked professionally with both Ruby and JavaScript and maintains an open source presence that goes back 12 years.

He was a member of the Pry core team from 2011 to 2019, he is a member of the webmachine-ruby team, and considers himself to be in a constant state of learning as well as being interested in all programming languages.

He has authored and maintained a diverse range of applications, from Rails applications to a Google Chrome extension all the way down to a node-webkit application with a Ruby backend.

My Projects

trip.rb

Trip is a concurrent tracer that can pause, resume and alter code while it is being traced. The tracer yields control between two threads, typically the main thread and a thread that Trip creates.

xchannel.rb

xchannel.rb provides an easy to use abstraction for sharing Ruby objects between Ruby processes who share a parent-child relationship.

xpool.rb

xpool.rb is a light weight in-memory process pool that was built with xchannel.rb. A process pool can utilise all CPU cores on CRuby, while also providing an isolated memory space for running a job.

tinyhttp.js

tinyhttp.js provides a promise-based API built around XMLHttpRequest. It was originally written for use inside the Private Internet Access browser extension that I helped to author.

rstl.js

Really Small Template Language (rstl.js) is a small function that implements a template language, it is similar to mustache.js but much smaller and with less features.
